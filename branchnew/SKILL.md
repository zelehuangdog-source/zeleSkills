description: Fork the current Claude Code session into a new iTerm split pane (to the right)
argument-hint: [name]
allowed-tools: Bash(branchnew *)
---
!`branchnew $ARGUMENTS`

The command above forked the current session into a new iTerm split pane on the
right (the fork's name appears in the output, if any). In ONE short line, tell
the user the fork is open. Do not call any tools and do not do anything else.
