# branchnew

> **Fork 来源**: [limin112/branchnew](https://github.com/limin112/branchnew)
> 
> 本 skill 基于 limin112 的原始实现 fork 而来，感谢原作者。

Fork the current Claude Code session into a new iTerm split pane (to the right).

## 用法

```
/branchnew [name]
```

会在 iTerm 右侧新开一个 split pane，将当前 Claude Code session fork 过去。

## 安装

将 `SKILL.md` 复制到 `~/.claude/commands/branchnew.md`。

## 前置依赖

- 需要安装 `branchnew` CLI 工具（见原始仓库：https://github.com/limin112/branchnew）
- iTerm2 终端
